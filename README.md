# Java-Collection-Framework-Concepts-and-Examples
This repository contains simple and practical Java programs demonstrating
the usage of the Java Collection Framework.

## Topics Covered
- List, Set, Map, Queue implementations
- HashMap, HashSet, TreeSet, IdentityHashMap
- ArrayDeque, PriorityQueue
- Iterators, Comparators, Generics
- Concurrent and Legacy collections

## Technologies Used
- Java
- Eclipse IDE

## Purpose
Created to strengthen understanding of core Java collection concepts
through hands-on examples.
